JavaScript (JS)

A scripting or programming language that allows you to implement complex features on web pages.

Note: Every time a web page does more than just sit there and display static information for you to look at — displaying timely content updates, interactive maps, animated 2D/3D graphics, scrolling video jukeboxes, etc. — you can bet that JavaScript is probably involved.

Operators

JavaScript operators are used to assign values, compare values, perform arithmetic operations, and more.

W3Schools.com
Variables

Variable means anything that can vary. JavaScript includes variables which hold the data value and it can be changed anytime.

TutorialsTeacher.com
Functions

A function is a JavaScript procedure—a set of statements that performs a task or calculates a value.

MDN
Arguments

Arguments is an Array-like object accessible inside functions that contains the values of the arguments passed to that function.

Note: “Array-like” means that arguments has a length property and properties indexed from zero, but it doesn't have Array's built-in methods like forEach() and map().
